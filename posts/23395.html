国内加速访问 GitHub

中国访问 GitHub 的速度不忍直视，那就叫一个慢！

解决方法

不需要科（）学（）上网，也不需要开代理加速器。

一、查询网址对应的 IP 地址

查询地址：https://www.ipaddress.com/

点击下面 域名（domain）自动跳转到 ipaddress.com 查询结果页面：

  1. github.com

  2. assets-cdn.github.com

  3. github.global.ssl.fastly.net

二、修改本地系统 hosts 文件

hosts 文件路径：C:\Windows\System32\drivers\etc\hosts

  1. 以管理员身份打开 CMD 终端，并输入 notepad ，会打开一个记事本。

  1. 打开文件，输入 hosts 文件路径，选择所有文件，最后点击 hosts 自动打开该文件。

  1. 修改 hosts 文件，将查询到的 IP 和 域名写在 hosts 文件的最后即可

三、刷新本地系统 dns 缓存

Windows 下利用 Git 生成 SSH KEY 并配置到 GitHub
基本环境 Windows 操作系统 本地安装了 Git 一个 GitHub 帐号 生成 SSH KEY打开 Git Bash在 Windows 桌面上，鼠标右击，选择 Git Bash here，打开 Git Bash 输入命令email@email.com 填入一个你自己的邮箱，可以是注册 GitHub 的邮箱 输入以下命令，一直按回车： 1ssh-keygen -t rsa -C "email@email.com" 查看 SSH KEY输入以下命令查看 SSH KEY： 1cat ~/.ssh/id_rsa.pub 将 SSH KEY 添加到 GitHub 点击头像，选择 Settings 选择 SHH and GPG keys 点击 New SSH key 填入 SSH KEY 点击 Add SSH key 测试连接打开 Git Bash，输入以下命令： 1ssh -T git@github.com
